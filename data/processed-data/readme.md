# processed-data

This folder contains cleaned and analysis-ready datasets generated by the 
cleaning pipeline in `code/mimic-processing.qmd`.

## Files
- `analysis_dataset.csv` — the main analytic dataset used in all analyses 
  and the manuscript. One row per hospital admission. Contains demographics, 
  30-day mortality outcome, and binary medication class flags.

## How it is generated
This file is automatically created by running `code/mimic-processing.qmd`. Do not edit 
it manually. To regenerate it, run:

    quarto render code\processing-code\mimic-processing.qmd
