# processing-code

This folder contains all processing scripts needed for the project

## Files

- `mimic-processing.qmd` —  — loads the raw MIMIC-IV CSV files, cleans and merges 
  tables, computes 30-day mortality, assigns medication classes, and saves 
  the final analytic dataset.

## Inputs

- `data\raw-data\admissions_syn.csv` 
- `data\raw-data\patient_syn.csv` 
- `data\raw-data\prescriptions_syn.csv` 

— generated by `data\raw-data\raw-synthetic-data.qmd`

## Outputs

- `mimic-processing.html` — rendered HTML report containing all data processing code

## How to Run

From the project root:

    quarto render code/eda-code/mimic-processing.qmd

## Notes

- The raw data script must be run before this script, as it depends on 
  `data\raw-data\raw-synthetic-data.qmd`
