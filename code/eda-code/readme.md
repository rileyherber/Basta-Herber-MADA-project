# eda-code

This folder contains all exploratory data analysis scripts for the project.

## Files

- `mimic-eda.qmd` — loads the processed dataset and produces all exploratory 
  plots and tables examining medication use and 30-day mortality.

## Inputs

- `data/processed-data/analysis_dataset.csv` — generated by `code/processing-code/mimic-processing.qmd`

## Outputs

- `mimic-eda.html` — rendered HTML report containing all exploratory plots and tables

## How to Run

From the project root:

    quarto render code/eda-code/mimic-eda.qmd

## Notes

- The processing script must be run before this script, as it depends on 
  `data/processed-data/analysis_dataset.csv`
- Plots produced here are used in the manuscript. If you change anything 
  in this script, re-render the manuscript afterwards.
